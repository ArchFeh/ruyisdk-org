ヒント

下载并使用 RuyiSDK，即表示您同意许可条款和隐私声明。

您可以从 GitHub releases 或 RuyiSDK 镜像中获取预构建的二进制文件

下载​

$ wget https://mirror.iscas.ac.cn/ruyisdk/ruyi/testing/ruyi.amd64.20231211  #从软件源下载ruyi本体
$ chmod +x ./ruyi.amd64.20231211  #给ruyi添加执行权限
$ sudo cp ruyi.amd64.20231211 /usr/local/bin/ruyi #将ruyi本体改名为ruyi，并放入PATH
$ ruyi version #成功！
